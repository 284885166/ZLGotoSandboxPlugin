ZLGotoSandboxPlugin-Xcode
------------

这是一款快速跳转Xcode沙盒的插件，能够时时监听模拟器改变/Xcode改变
-----
#### 快捷键 'Shift + Common + w' (当模拟器运行应用的时候有效,中文不支持快速跳转)
#### 运行完程序后需要重启下Xcode
